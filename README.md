## :scroll: QuickStart

1. Clone this repo or download the latest release. Rename it to whichever project name you want (Must have Docker desktop installed).
2. `cd <K8s-lab> <lab-name>`
3. `pnpm install`
4. `pnpm test | pnpm test:watch`
5. **YOU ARE READY TO GO**
